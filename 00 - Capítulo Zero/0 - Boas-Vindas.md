# Boas-Vindas!

### Este é o ínicio da sua jornada! Este curso é feito para satisfazer todos os níveis de conhecimentos para auxiliar no contato com a primeira linguagem de programação. Vamos juntos!

Neste caminho, você vai:

- Entender como um computador funciona

- Desenvolver raciocínio lógico

- Dar seus primeiros passos na programação em C

- Construir bases sólidas para seguir adiante

Nosso objetivo é simples: te ajudar a aprender, de forma clara, prática e sem pressa.

### O curso segue a seguinte estrutura:

- Capítulos: cada uma das pastas do repositório é um conteúdo e cumpre um propósito. Se você já domina o conteúdo, pode pular para o próximo capítulo. Caso queira verificar sua capacidade, faça o teste ao final do capítulo e obtenha 80% de acerto para passar para o próximo capítulo.

- Lições: dentro de cada capítulo temos várias lições. Cada lição tem um objeto de ensino específico. Recomendamos fazer as lições em ordem.

- Testes: as lições podem conter testes, que serão importantes para avaliar seu entendimento sobre o conteúdo que está sendo ensinado. Preste bastante atenção nos testes, pois eles serão essenciais para consolidar seu entendimentos. Você pode voltar para refazer os testes quando quiser, em qualquer parte da sua jornada.

- Bisus: os bisus são resumos de todo o conteúdo do capítulo. Use para revisões e busca rápida de informações. Para fazer o melhor uso dos bisus, é essencial que você tenha pleno domínio do conteúdo do capítulo.
